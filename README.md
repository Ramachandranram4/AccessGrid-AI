# AccessGrid AI

AI-powered realtime accessibility navigation platform designed to help visually impaired citizens, wheelchair users, and elderly people navigate cities more safely, independently, and efficiently.

---

## 🚀 Inspiration

In many cities like Tijuana, daily mobility remains a major challenge for accessibility-challenged communities. Damaged sidewalks, blocked ramps, inaccessible crossings, and unsafe pathways make even short journeys stressful and dangerous.

Traditional navigation systems optimize only for distance and speed — but they ignore accessibility.

AccessGrid AI was created to solve this problem using AI-powered realtime accessibility guidance, safer route navigation, and community-driven accessibility intelligence.

---

# 🌍 Problem Statement

People with:
- Visual impairments
- Wheelchair accessibility needs
- Elderly mobility limitations

often face:
- Blocked wheelchair ramps
- Broken sidewalks
- Unsafe crossings
- Construction barriers
- Inaccessible public pathways

Current navigation systems do not adapt to accessibility conditions in realtime.

---

# 💡 Solution

AccessGrid AI acts as a personal realtime mobility companion.

The platform continuously:
- Detects surroundings
- Analyzes accessibility conditions
- Guides users using AI voice assistance
- Provides safer route recommendations
- Detects obstacles in realtime
- Updates accessibility intelligence dynamically

Users can also report hazards to help create a living accessibility map for the community.
